Jarvis is a Python-based AI voice and text assistant that can browse websites, play music, search the web, fetch news, and answer questions using Google’s Gemini API. It combines speech recognition, text-to-speech, web automation, and AI-powered chat for a versatile personal assistant experience.

Features

Voice & Text Input – Interact via typing or speaking.

Website Automation – Open popular sites like Google, YouTube, Gmail, Instagram, LinkedIn, and more.

Search & YouTube Playback – Search Google or play songs directly from YouTube or YouTube Music.

News Fetching – Get the latest headlines.

AI Chat – Converse naturally using Google Gemini API.

Responsive Commands – Recognizes “Jarvis” and responds.
